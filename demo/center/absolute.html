绝对居中

固定宽
 .absolute-center { position: absolute; margin: auto; top: 0; bottom: 0; left: 0; right: 0; }

负外边距居中

negative
 .negative-margin { position: absolute; left: 50%; top: 50%; margin-top: -50px; margin-left: -50px; }

CSS变换居中(transform)

transform
 .transform-center { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }

表格居中

table
 .table-center { display: table; } .table-cell { display: table-cell; vertical-align: middle; } .center-content { display: table; margin: auto; }

行内块级元素居中

inline block
 .inline-block-center { text-align: center; } .inline-block-center:before { content: ''; display: inline-block; height: 100%; vertical-align: middle; margin-left: -0.25em; } .inline-block-content { display: inline-block; vertical-align: middle; width: 50%; }
